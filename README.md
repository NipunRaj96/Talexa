# Talexa

**The AI-Powered Recruitment Platform**

Talexa streamlines the hiring process by using advanced AI to analyze resumes, match candidates to job descriptions, and provide actionable insights for recruiters.

## ‚ú® Key Features

-   **AI Resume Analysis**: Automatically extracts skills, experience, and education from PDF/DOCX resumes.
-   **Smart Matching**: Scores candidates against job requirements using LLMs (Llama 3.3).
-   **Recruiter Dashboard**: A centralized hub to manage job postings and view applicant insights.
-   **Secure Authentication**: Enterprise-grade security with Google OAuth and Magic Links.
-   **Modern Interface**: Built with React, Tailwind CSS, and Framer Motion for a premium user experience.

If you are a developer looking to contribute or run Talexa locally:

1.  Clone the repository.
2.  Install dependencies for `frontend` (Node.js) and `backend` (Python).
3.  Configure your environment variables (Supabase & Groq keys required).
4.  Run the development servers.
